# Verstal Landing

Лендинг продукта Verstal: единая агентная платформа для разработки и управления рабочим местом.

Главная идея страницы:

> Core бесплатный. Go даёт модели из коробки. Pro даёт управление рабочим контуром. Deploy даёт внедрение под ключ.

Ключевое усиление hero: без VPN, оплата по расчётному счёту, соблюдение 152-ФЗ.

## Структура

```text
verstak/
├── index.html
├── README.md
└── assets/
    ├── logo.svg
    ├── logo-*.png

```

## Запуск локально

```bash
cd landings/verstak
python -m http.server 8000
```

Открыть: `http://localhost:8000`

Можно также открыть `index.html` напрямую в браузере.

## Секции

- Hero: “единая агентная платформа для разработки и управления рабочим местом”.
- Product mock: проекты, модели, дифф, approval, журнал.
- Lineup: Core / Go / Pro / Deploy.
- Workflow: проект → модель → действие → контроль → отчёт.
- Why pay: какая боль снимается на каждом слое.
- Honest limits: чем Verstal пока не побеждает Cursor/Codex.
- Deploy: внедрение под B2B и клиентский контур.

Не использовать:

- “русский Cursor”
- “любая модель, один IDE”
- “модели РФ лучше для кода”

Использовать:

- “единая агентная платформа”
- “без VPN, оплата по расчётному счёту, соблюдение 152-ФЗ”
- “модель в комплекте или твоя подписка”
- “дифф перед применением”
- “журнал действий по клиентскому проекту”
- “Core / Go / Pro / Deploy”
