## Твоя часть — Фаза 2 (зона `src/`)

**UI-pill из подсказки.** Сейчас подсказка приходит обычным info-событием (текст в чате). Сделай из неё **кликабельный pill** в композере/Timeline:
- Триггер: пришёл `info`-event с текстом, начинающимся на `🔧 Похоже на терминальную задачу`.
- Вид: pill «🔧 Терминальная задача → Claude Code?» рядом с композером.
- Клик: переключает провайдера текущего чата на сконфигурированный CLI (Claude Code) через `setProviderModel` (как ручной свитч в селекторе).
- **Гейт:** показывать pill только если CLI-провайдер реально настроен у пользователя (иначе не предлагать инструмент, которого нет).

## Живая проверка (прокликать в запущенном app)
1. **Сценарий А (бесшовность):** посреди чата переключи провайдер API→CLI→API. Убедись, что история и контекст не теряются (нить разговора цела).
2. **Фаза 3 (делегирование):** дай команду на терминальную задачу («почему падает сборка tsc, исправь и прогони тесты») и через `delegate_task` делегируй на Claude Code (`provider_id=claude-cli`). Убедись, что CLI отрабатывает и результат виден в Timeline как `subagent-run`.
