### Verstak 2.0.6 — проекты, просмотр файлов в чате + важные исправления

Два блока: большое обновление проектов и чата, и три критичных исправления, которые незаметно ломали работу.

**📁 Проекты**
- Полностью переработанные настройки проекта: заметки, ярлыки, статус (активный / на паузе / завершённый), цвет проекта, управление уведомлениями, данные проекта.
- Ярлыки — для фильтрации и организации клиентов. Цвет проекта виден вокруг аватарки, когда в проекте нет активной работы.
- Копирование проекта с сохранением настроек. Безопасная очистка временных данных — без удаления чатов, задач и файлов.
- Левое меню: фильтры по статусам и архиву, панель «Мои проекты» расширяется по ширине, папки раскрываются прямо из свёрнутого меню.

**💬 Просмотр файлов прямо из чата**
- Путь к файлу открывается в правой панели: текст, Markdown, код, документы Word, таблицы Excel. Для нечитаемых файлов — понятное объяснение вместо технической ошибки.
- Блок «Текст для копирования»: модель присылает текст в рамке, копируемый одной кнопкой.
- Автопрокрутка больше не «прыгает» вниз, если вы ушли выше по истории и пишете сообщение.

**⚡ Быстрее**
- Набор текста в длинных чатах, переключение между проектами, список проектов на больших базах.

**🔴 Критичные исправления**
- **Модели Claude в Настройках были мертвы.** В мае Anthropic переименовал модели; исправление тогда затронуло движок, но **вкладку Настроек забыли** — почти два месяца она предлагала три несуществующие модели, а актуальную (`claude-sonnet-4-6`) выбрать было нельзя вообще.
- **Провайдер Codex OAuth не выбирался** — молча подменялся на Gemini.
- **Подписка Codex могла «выкинуть» из аккаунта:** если при обновлении токена его не удавалось сохранить на диск, доступ терялся. Теперь сессия продолжает работать, запись повторяется, а при неудаче приложение честно предупреждает. Попутно: два параллельных чата больше не ломают токен друг другу, токен не остаётся во временном файле.

**🐛 Прочее**
- Быстрое переключение между чатами могло применить настройку модели не к тому чату.
- Списки моделей Verstak Gateway и OpenRouter в Настройках были неполными.
- Таймаут долгих задач увеличен до 90 минут. Улучшено восстановление ярлыков после обновления.

**🛡️ Под капотом**
Добавлены автоматические сторожа: линтер на изменённые файлы перед коммитом, страж расхождения списков моделей между интерфейсом и движком (именно он поймал мёртвые модели Claude), состязательные тесты гонок и отказов авторизации. Релиз собирается из закоммиченного кода и проходит объективный гейт перед публикацией.

Обновление придёт автоматически. Полный список — в `CHANGELOG.md`.
