### Контролируемый AI-дежурный + бандл надёжности и безопасности (1.9.6–1.9.9)

Первое обновление после 1.9.5 — одним махом доезжает всё накопленное: три закрытые арки усиления (доверие, длинные сессии, чистка ядра) плюс новая продуктовая фича.

**🕒 Новое: «Контролируемый AI-дежурный»**
- **Плановый дозор с готовыми сценариями.** Вкладка «Расписание» — сценарии в один клик: 🔌 Дозор интеграций (свежие данные Ozon / Wildberries / Яндекс.Метрики / 1С / Битрикс24 → аномалии), 🔎 Дозор кода (свежие TODO/FIXME и слабые места → с указанием файл:строка), 📊 Утренняя сводка данных (метрики за вчера против позавчера). Фоновый прогон строго только читает — ничего не меняет и не выполняет команд.
- **Мост «Починить в чате».** Находка ночного дозора теперь видна прямо в списке задач и одной кнопкой превращается в контролируемый разбор в чате: агент ищет первопричину, показывает источник и предлагает план — а правки и команды идут только под вашим подтверждением. Робот наблюдает — чинит человек, под контролем.

**🛡 Доверие и безопасность (1.9.6)**
- Откат правок CLI-прогона прямо из интерфейса (Инспектор запусков → «↩︎ Откатить», с предпросмотром что изменится).
- Честная маркировка защиты секретов по каждому CLI: приложение больше не выглядит защищённее, чем есть на самом деле.
- Закрыт путь утечки: inline-токены в аргументах команд (`Authorization`, `git remote https://user:pass@`, `?token=`) больше не попадают в интерфейс сырыми.

**⏱ Длинные сессии (1.9.7)**
- Прерванный CLI-прогон больше не тупик — повтор запроса одной кнопкой.
- Авто-переключение подписочного аккаунта на лимите теперь работает и для CLI-провайдеров (раньше срабатывало только на API-пути).
- Меньше износа диска и надёжнее точка отката на долгих прогонах.

**🧱 Под капотом (1.9.8)**
- Возобновление после сбоя больше не путает провайдеров (история прогона переигрывается только на совместимом провайдере).
- Секреты не текут в логи по значению (редакция чистит и содержимое полей, не только по имени ключа).
- Крупная внутренняя перестройка ядра агента — без изменения поведения, проверено тестами; база готова к дальнейшему развитию.

**🩹 Исправлено**
- Индикатор «идёт ответ» больше не залипает у фонового чата при уходе в справку.

Обновление придёт автоматически. Полный список изменений — в `CHANGELOG.md`.
