### Verstak 2.0.5 — UI-рефреш + дневной лимит расходов + фиксы коннекторов

**🎨 Обновлённый интерфейс (пакет Ильи)**
- Переработанные **Настройки**: новый вид коннекторов с иконками сервисов, аккуратнее сгруппированные разделы.
- **Всплывающие подсказки** по всему интерфейсу — единый механизм, понятнее назначение элементов.
- Новая вкладка **«Правила проекта»** — правила работы агента видны и редактируются прямо в приложении.
- Доработки тем и оформления.

**💰 Дневной лимит расходов**
- Cost-cap теперь считается **за сутки** и переживает перезапуск: было — за сессию (обнулялся при закрытии), стало — лимит `$/сутки`, накопленная за день сумма сохраняется, сбрасывается на новый день. В статус-баре — реальный дневной расход.

**🐛 Исправления**
- Доработки коннекторов Битрикс24 (профиль компании) и Яндекс.Wordstat.

**Без изменений:** провайдер Codex OAuth (Experimental) и 3-режимная таксономия (Туннель · CLI · API) из 2.0.4 сохранены полностью.

Обновление придёт автоматически. Полный список — в `CHANGELOG.md`.
