### Verstak 2.0.5 — большой UI-рефреш + суточный лимит расходов

Крупное обновление интерфейса поверх 2.0.4. Провайдер **Codex OAuth (Experimental)** и 3-режимная таксономия (Туннель · CLI · API) из 2.0.4 сохранены полностью.

**⚙️ Настройки**
- Полностью переработано меню: единый стиль, компактные карточки, важные настройки раскрываются внутри выбранного блока.
- «Внешний вид» проще: понятные режимы темы, плотности, масштаба и анимаций.
- «Уведомления»: общее вкл/выкл, тихие часы, режимы показа, тестовое уведомление.
- «Обновления»: патчноуты после обновления, безопасная очистка временных файлов, понятные состояния проверки.
- «Профиль» временно закрыт как будущий раздел (аккаунт, организация, команда).

**🧠 Модели и провайдеры**
- Понятнее отображение подключённого, настройки прямо внутри карточки, человеческие описания без техножаргона.
- Подсказки: какие модели контролируются Verstak, а какие работают как внешние агенты и могут не подчиняться всем правилам.
- **Лимит расходов теперь суточный** и привязан к локальному дню (переживает перезапуск).
- Убраны лишние и дублирующие плашки статусов.

**🔌 Коннекторы**
- Переработанные карточки: подключённые/неподключённые разделены, статусы нагляднее, настройки внутри карточки.
- Bitrix24 — видно, какие разделы доступны по текущему вебхуку. Wordstat — подключение через Yandex Search API с понятными полями и проверкой ключа.
- Иконки в стиле Verstak. Сохранение неверного ключа больше не показывает коннектор подключённым.

**🧩 Внешние инструменты**
- Раздел MCP переименован в «Внешние инструменты» / «Продвинутые подключения» с понятными описаниями.

**🛡️ Права модели**
- «Что разрешено» → «Права модели»: компактная карта режимов (что модель делает сама, где нужно подтверждение, что заблокировано) + предупреждение о невозможности полного контроля внешних моделей.

**📁 Проекты и инструкции AI**
- «Память» и «Правила» перенесены в разделы конкретного проекта.
- Новый раздел проектных инструкций AI с готовыми блоками (базовый проект, маркетинг, разработка, поддержка, отчёты, операции) и понятным редактором.

**🎨 Интерфейс проекта**
- Обновлён дизайн карточек, кнопок, полей, подсказок; папки проектов раскрываются из боковой панели; улучшены лампочки статусов в обеих темах.

Обновление придёт автоматически. Полный список — в `CHANGELOG.md`.
