# Черновик патчноута

## Проекты

- Полностью переработаны настройки проекта: добавлены заметки, ярлыки, статус проекта, цвет проекта, управление уведомлениями, данные проекта и отдельный блок управления проектом
- В проект можно добавить ярлыки, чтобы позже использовать их для фильтрации и организации клиентов
- Добавлены статусы проекта: активный, на паузе и завершённый
- Добавлен цвет проекта: он отображается вокруг аватарки, когда проект не открыт и в нём нет активной работы, ошибки или нового ответа
- Добавлено копирование проекта с сохранением его настроек
- Добавлена безопасная очистка временных данных проекта без удаления чатов, задач и файлов проекта
- В левом меню проектов добавлены фильтры по статусам и архиву
- Панель "Мои проекты" теперь можно расширять по ширине
- В свёрнутом меню проектов папки можно раскрывать и скрывать прямо из боковой панели

## Чат

- Добавлен просмотр файлов прямо из чата: путь к файлу можно открыть в правой панели, не переходя в проводник
- В панели просмотра теперь можно читать текстовые файлы, Markdown, код, документы Word и таблицы Excel
- Для файлов, которые нельзя открыть, показывается понятное объяснение вместо технической ошибки
- Добавлен отдельный блок "Текст для копирования": модель может отправить текст в рамке, откуда его можно скопировать одной кнопкой вместе с нумерацией
- Улучшена автопрокрутка: если пользователь вручную ушёл выше по чату и пишет сообщение, чат не должен сам прыгать вниз до отправки

## Производительность

- Ускорен набор текста в длинных чатах: предпросмотр токенов больше не нагружает историю сообщений во время печати
- Ускорено переключение между проектами: дерево файлов подгружается только при открытии вкладки файлов
- Оптимизирован список проектов, чтобы левое меню быстрее работало на больших базах

## Работа AI

- Таймаут долгих задач увеличен до 90 минут, чтобы большие задачи не останавливались слишком рано

## Запуск приложения

- Улучшено восстановление ярлыков Verstak на рабочем столе, в меню Пуск и на панели задач
- После локального обновления ярлыки должны снова указывать на установленный Verstak, а не на временные или резервные файлы
