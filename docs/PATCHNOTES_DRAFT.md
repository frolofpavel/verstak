# Черновик патчноута

## Чат

- Улучшена работа длинных переписок: старые сообщения можно подгружать отдельно, а новые чаты открываются быстрее
- Исправлена автопрокрутка: если пользователь читает сообщения выше и пишет текст, чат не должен сам прыгать вниз
- Исправлена кнопка остановки ответа: она должна останавливать текущую работу модели, не задевая другие проекты
- Добавлен надёжный блок текста для копирования: модель может отправить текст в отдельной рамке, а пользователь скопирует его одной кнопкой вместе с нумерацией
- Улучшено копирование из блоков кода и текстовых рамок

## Файлы

- Улучшен просмотр файлов из чата: пути к файлам открываются в боковой панели, если файл доступен в проекте
- Для файлов, которые нельзя открыть, показывается понятное объяснение вместо сырой технической ошибки
- Тяжёлые папки в дереве файлов открываются аккуратнее и меньше нагружают интерфейс

## Управление проектом

- Боковая панель проекта переименована в "Управление проектом", чтобы не путаться с панелью "Мои проекты"
- Разделы "Браузер" и "Дизайн" временно закрыты плашкой "Скоро"

## Уведомления

- Всплывающие уведомления больше не должны перекрывать невидимой областью остальной интерфейс

## Работа AI

- Длинные задачи могут выполняться до 90 минут
- Остановка ответа стала надёжнее для внешних и консольных моделей
